# Simple computer architecture

## Build
```
make
```

## Translate Simple Basic to Simple Assembler
```
./simplebasic/sbt <in.sb> <out.sa>
```

## Translate Simple Assembler to machine code
```
./simpleassembler/sat <in.sa> <out.o>
```

## Run Simple computer
```
./console/sc
```
